About Me

Hi, I'm Zeke. I'm a software developer with skills in React, Redux, Ruby, Rails, JavaScript, and Node.js. I'm passionate about building data-driven web applications with excellent UI and a strong practical purpose for end-users. As an example, I recently led a team building a web app called Kinship that allows users to store and organize information about friends, family, and social contacts.

I also enjoy technical challenges and learning how to use new tools and libraries. I recently created a full-scale clone of Airbnb that integrated the Google Maps API and the React-Dates calendar picker library.

Previously I worked as an editor in the financial industry and a journalist covering Russia.
